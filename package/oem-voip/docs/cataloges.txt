	Ниже будет дано описание структуры каталогов и файлов проекта. Проект для
компиляции и сборки использует OpenWRT и разрабатывается в качестве пакета.
Реализована также возможность собирать некоторые программые модули проекта 
отдельно, не собирая весь проект (описание в build_system.txt). Если
оставить только основное, то выглядит эта структура каталогов проекта так:

../
|-- Config.in
|-- Makefile
|-- README
|-- docs
|-- files
|   |-- fw
|   |-- patches
|   |-- rc
|   `-- src
|       |-- build.sh
|       |-- drv_daa-1.0.2.0.tar.gz
|       |-- drv_sgatab
|       |-- drv_tapi-3.6.1.tar.gz
|       |-- drv_vinetic-1.3.1_tapi-3.6.1.tar.gz
|       |-- itmp
|       |-- libab
|       |-- libconfig
|       |-- svc
|       |-- svd
|       |-- svi
|       `-- tst
`-- ipkg

Здесь:

|-- Config.in
	Является частью системы настройки OpenWRT и определяет переменные,
идентифицируемые конфигуратором и определяющие включён пакет в сборку прошивки
или нет.

|-- Makefile
	Является частью системы сборки OpenWRT и содержит инструкции по сборке
проекта.

|-- README
	Главный файл описания проекта, в котором даются ссылки на более детальное
описание.

|-- docs
	Документация по проекту.

|-- files
	Основные файлы проекта.

|   |-- fw
	Прошивка для VoIP-чипов Vinetic-2CPE, загружаемая после старта системы
программой svi (см. progs.txt).

|   |-- patches
	Исправления для стека драйверов Vinetic-a (см. progs.txt).

|   |-- rc
	Скрипт для запуска и остановки VoIP-демона svd (см. progs.txt).

|   `-- src
	Исходные коды проекта.

|       |-- build.sh
	Скрипт для ручной сборки отдельных программных элементов пакета (драйверы,
svd, svi), a также для изменения патчей к стеку драйверов. (см.
build_system.txt).

|       |-- drv_daa-1.0.2.0.tar.gz
	Архив DAA-драйвера от Infineon (см. progs.txt).

|       |-- drv_sgatab
	Драйвер для управления материнскими VoIP-платами (см. progs.txt).

|       |-- drv_tapi-3.6.1.tar.gz
	Архив высокоуровнего TAPI-драйвера от Infineon (см. progs.txt).

|       |-- drv_vinetic-1.3.1_tapi-3.6.1.tar.gz
	Архив низкоуровнего VINETIC-драйвера от Infineon (см. progs.txt).

|       |-- itmp
	Здесь происходит распаковкаи сборка стека драйверов от Infineon при ручной
сборке (см. build_system.txt).

|       |-- libab
	Библиотека обёртка (wrapper) для обеспечения высокоуровнего доступа к
VoIP-оборудованию (см. progs.txt).

|       |-- libconfig
	Библиотека для чтения файлов конфигурации. Используется в svi и svd (см.
progs.txt).

|       |-- svc
	Программа очистки после сбоя svd. Если была запущена генерация rtp-трафика
и по каким-либо причинам произошёл сбой svd, эта программа выполнит
необходимое закрытие каналов и останов генерации. (см. progs.txt).

|       |-- svd
	VoIP-демон и интерфейсная программа к нему.(см. progs.txt).

|       |-- svi
	Инициализирующая программа. Загружает прошивку в VoIP-чипы, стек драйверов
и производит инициализацию каналов (см. progs.txt).

|       `-- tst
	Набор программ для тестирования VoIP-оборудования (см. progs.txt).

`-- ipkg
	Описание пакета для OpenWRT.
